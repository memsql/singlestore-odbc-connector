SingleStore ODBC Driver files will be installed in folder /Library/SingleStore/SingleStore-Connector-ODBC.

Important: The driver library requires openssl and libiodbc libraries, that are not available natively, and can be installed with Homebrew.

Homebrew installation instructions can be found here. The short version at the moment is to run the following command in the terminal:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

The command to install dependecies is:

brew install openssl@1.1 libiodbc
